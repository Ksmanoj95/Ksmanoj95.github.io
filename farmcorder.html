Manoj Kumar

FARMCORDER

A Low cost Agro Sensor

Farming remains to be backbone of developing countries like India and Africa and it is the principal source of livelihood for majority of the population in the countries. Farmers however use antiquated techniques which lead to overuse or under usage of fertiliser,water which leads to disastrous yields.

Our solution to this problem is “FarmCorder”. We propose a new simple and intuitive sensor based networked system which consist of base hand held module which consists of a CC3200 WIFI MCU and a modular I2C based sensor 7 pin connector interface, MTK33339 GPS,Rechargeable battery,Solar charging/Crank charging, Low power SHARP memory display.

The sensor interface uses the I2C interface to connect to replaceable sensory devices such as
i)Leaf Spectroscopy Clip
ii)Dielectric Soil Moisture,Temperature sensor
iii)Soil Testing system (Color analysis)

The sensor data along with the GPS data are sent to a server using the CC3200 WiFi using a local hotspot and compared with a preexisting database.The farmer is then provided with a comprehensive fertilizer application plan via IVRS(Interactive Voice Response) in his/her local language.
